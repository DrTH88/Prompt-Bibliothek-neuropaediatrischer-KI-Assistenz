# Prompt für administrative Tätigkeiten: Gutachten, Anträge, Hilfsmittel, Pflegegrad, GdB, Reha, Schulbegleitung

```text
Unterstütze mich bei der Erstellung eines medizinisch fundierten administrativen Dokuments für ein Kind oder einen Jugendlichen mit neuropädiatrischer / sozialpädiatrischer Erkrankung.

Dokumenttyp:
[Gutachten / ärztliche Stellungnahme / Antrag Hilfsmittel / Antrag Heilmittel / Reha-Antrag / Pflegegrad-Stellungnahme / GdB-Stellungnahme / Schulbegleitung / Nachteilsausgleich / Eingliederungshilfe / Widerspruch / Kostenübernahme / Sonstiges]

Adressat:
[Krankenkasse / MD / Rentenversicherung / Sozialamt / Jugendamt / Schule / Versorgungsamt / Gericht / andere Stelle]

Patient, anonymisiert:
- Alter:
- Diagnose(n):
- Verdachtsdiagnosen:
- Funktionsbeeinträchtigungen:
- Entwicklungsstand:
- Mobilität:
- Kommunikation:
- Kognition:
- Verhalten / Emotion:
- Epilepsie / Anfälle:
- Ernährung / Schlucken:
- Schlaf:
- Pflegebedarf:
- Teilhabe:
- Kindergarten / Schule:
- Therapien:
- Hilfsmittel:
- bisherige Maßnahmen:
- aktuelles Problem:
- beantragte Leistung:
- Begründungsziel:

Aufgabe:
Erstelle eine professionelle ärztliche Stellungnahme in sachlicher, klarer und überzeugender Sprache. Der Text soll medizinisch begründet, nachvollziehbar und formal geeignet sein. Er soll keine juristische Beratung vortäuschen, aber medizinische Argumente so formulieren, dass die beantragte Leistung plausibel begründet ist.

Wichtige Regeln:
1. Verwende ausschließlich die von mir angegebenen Fakten.
2. Erfinde keine Diagnosen, Befunde, Behinderungsgrade, Pflegezeiten, Scores, Gesetzesgrundlagen oder Vorbefunde.
3. Wenn Angaben fehlen, markiere sie unter „Für eine belastbare Stellungnahme fehlen noch“.
4. Formuliere funktionsorientiert:
   - Was kann das Kind nicht oder nur eingeschränkt?
   - Welche Unterstützung ist erforderlich?
   - Welche Risiken entstehen ohne Unterstützung?
   - Welche Teilhabe wird durch die Maßnahme ermöglicht?
5. Nutze ICF-orientierte Sprache, soweit passend:
   - Körperfunktionen
   - Aktivitäten
   - Teilhabe
   - Umweltfaktoren
   - personbezogene Faktoren
6. Stelle den Zusammenhang zwischen Diagnose, Funktionsbeeinträchtigung und beantragter Leistung explizit her.
7. Beschreibe die Erforderlichkeit, Geeignetheit und Verhältnismäßigkeit der Maßnahme.
8. Verwende keine übertriebene oder emotionalisierende Sprache.
9. Vermeide pauschale Formulierungen wie „dringend erforderlich“, ohne konkrete medizinische Begründung.
10. Falls gesetzliche Grundlagen genannt werden, nur mit überprüfbarer Quelle oder als „bitte juristisch/formal prüfen“ kennzeichnen.
11. Wenn es sich um einen Widerspruch handelt:
   - fasse die Ablehnungsbegründung zusammen
   - entkräfte sie medizinisch
   - benenne fehlende Berücksichtigung von Funktionsbeeinträchtigungen
   - formuliere eine klare erneute Empfehlung
12. Schreibe so, dass der Text von ärztlicher Seite plausibel unterzeichnet werden könnte, aber prüfe am Ende auf fehlende Daten und Risiken.

Gewünschte Struktur:

1. Betreff
2. Anlass der Stellungnahme
3. Diagnosen und medizinischer Hintergrund
4. Aktueller Funktionsstatus
5. Auswirkungen auf Alltag, Pflege, Entwicklung, Sicherheit und Teilhabe
6. Bisherige Maßnahmen und deren Effekt
7. Begründung der beantragten Leistung
8. Folgen bei Nichtbewilligung
9. Ärztliche Empfehlung
10. Offene Angaben / bitte ergänzen
11. Optional: kurze Version für Formularfeld

Zusätzlich:
Erstelle nach dem Haupttext eine zweite, sehr knappe Version mit maximal 1.500 Zeichen für Online- oder Formularfelder.
```
